EP.1 มาแล้วครับ! 🎉 ตามสัญญาจากซีรีส์ "Coding Agent สำหรับทุกคน" รอบนี้เราจะมาเจาะลึกปูพื้นฐานเรื่อง LLM, Agent และ ReAct loop กันครับ

ความพิเศษของตอนนี้คือผมไม่ได้เขียนแค่บทความนิ่งๆ แต่จัดเต็ม Interactive Widgets มาให้ลองเล่นกันหน้าเว็บเลย! จะได้เห็นภาพชัดๆ ว่าไอ้ที่บอกว่า Agent มันคิดเองทำเองได้เนี่ย มันหมุน Loop ยังไง? แล้ว Chatbot ธรรมดากับ Agent มันต่างกันแค่ไหน กดดูความต่างแบบ Live ได้เลยครับ

แถมรอบนี้มี Bonus พิเศษ! ผมเขียนบทความแยกเรื่อง "Token" ไว้ให้ด้วย สำหรับใครที่อยากรู้ว่า AI มันย่อยภาษาที่เราพิมพ์เข้าไปยังไง ในนั้นมีเครื่องมือ Tokenizer ให้ลองจิ้มเล่นดูด้วยครับว่าประโยคที่เราพิมพ์ AI มันมองเป็นกี่ Token
